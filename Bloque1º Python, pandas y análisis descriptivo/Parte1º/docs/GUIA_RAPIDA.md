# Guía Rápida - Bloque 1 Parte 1

## 🎯 Objetivos de Aprendizaje

1. Dominar conceptos fundamentales de Python
2. Aprender manipulación de datos con pandas
3. Realizar análisis descriptivo de conjuntos de datos

## 📚 Estructura de Contenidos

### Módulo 1: Fundamentos de Python
- Variables y tipos de datos
- Estructuras de control
- Funciones y módulos

### Módulo 2: Introducción a Pandas
- Series y DataFrames
- Carga y exploración de datos
- Manipulación básica

### Módulo 3: Análisis Descriptivo
- Estadísticas descriptivas
- Visualización de datos
- Interpretación de resultados

## ⏱️ Temporalización Sugerida

- **Semana 1**: Fundamentos de Python (10-12 horas)
- **Semana 2**: Introducción a Pandas (10-12 horas)
- **Semana 3**: Análisis Descriptivo (8-10 horas)

## 💡 Sugerencias Pedagógicas

1. Ejecutar los notebooks de forma interactiva
2. Modificar el código y experimentar
3. Practicar con los datasets proporcionados
4. Resolver los ejercicios propuestos

## 📊 Datos Disponibles

Los datasets se encuentran en la carpeta `data/` y son reproducibles.

## ✅ Criterios de Éxito

Al finalizar esta parte, el estudiante debe ser capaz de:
- Escribir scripts Python funcionales
- Manipular dataframes con pandas
- Calcular y interpretar medidas descriptivas
- Crear visualizaciones básicas
