# Notebooks

Esta carpeta contiene los notebooks de Jupyter con los ejercicios y lecciones.

## 📌 Contenidos

Los notebooks estarán organizados de la siguiente manera:
- `XX_titulo_notebook.ipynb`: Enumerados por orden de ejecución

## 🚀 Cómo usar

1. Abre el notebook en Jupyter
2. Ejecuta las celdas secuencialmente
3. Experimenta con el código
4. Completa los ejercicios propuestos
